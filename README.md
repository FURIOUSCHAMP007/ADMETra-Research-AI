# ADMETra Research AI 🧪
### Advanced Pharmacological Analysis & Hazard Ranking Engine

ADMETra (Absorption, Distribution, Metabolism, Excretion, and Toxicity Research AI) is a next-generation laboratory dashboard designed for early-stage pharmacological screening. It combines deterministic molecular physics with generative artificial intelligence to predict the safety and efficiency profiles of chemical compounds.

---

## 🚀 Core Technologies

- **Frontend**: React 18 with Vite and TypeScript.
- **Styling**: Tailwind CSS for high-density research UI.
- **Analytics Engine**:
  - **RDKit (Minimal)**: Local physics-based computation for Molecular Weight, LogP, TPSA, and Lipinski rules.
  - **Google Gemini AI**: Deep semantic analysis for functional group risks, ADME profiling, and toxicity prediction.
- **Visualization**: Recharts for Pareto-frontier mapping and multi-objective scatter analysis.
- **Animations**: Framer Motion for sophisticated state transitions.

---

## ✨ Key Features

### 1. Dual-Path Analysis Pipeline
Every molecule undergoes a two-stage verification process:
- **Phase 1 (Physics)**: Immediate computation of physiological properties using RDKit logic.
- **Phase 2 (Intelligence)**: Background enrichment using Gemini 1.5 Pro to identify subtle biological liabilities.

### 2. Multi-Objective Ranking Engine
The **Ranking Engine** prioritizes compounds based on a weighted composite score:
- **Efficiency Index**: High absorption and permeability scores.
- **Hazard Index**: Penalization based on structural toxicity risks.
- **Safety Overrides**: Built-in critical hazard detection for high-profile substances like Fentanyl, Heroin, and Methamphetamine.

### 3. Visual Discovery Tools
- **Pareto Map**: Identifies the "golden mean" between bioavailability and toxicity.
- **Composite Rankings**: deduplicated leaderboard of optimal vs. hazardous candidates.
- **Explorer API**: Deep dive into individual molecular properties and Lipinski compliance.

### 4. AI-Powered Care Plans
Generates personalized safety protocols for each analyzed substance, including:
- Primary medical use cases.
- Lifestyle and interaction warnings (e.g., alcohol exclusion).
- Critical monitoring parameters (respiratory rate, metabolic load).

## 🛠️ Project Structure

- `/src/services/chemistry.ts`: The central logic for score calculation and RDKit property normalization.
- `/src/services/AnalysisContext.tsx`: Manages the complex state of compound batches, deduping, and AI enrichment.
- `/src/pages/Ranking.tsx`: High-contrast ranking interface with safety pulse indicators.
- `/src/data/drugs.json`: Curated seed database of common pharmacological agents.

## 🚦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Environment Setup**:
   Ensure a `GEMINI_API_KEY` is present in your environment if executing deep scans.
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
